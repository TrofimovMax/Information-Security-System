# Проект: Разработка ИС по обеспечению информационной безопасности

Подсистемы ИС:
- Портал
- Анализ исходного кода
- Анализ файла
- Проведение расследований
- Подсистема работы с инцидентами
- Подсистема работы с уязвимостями

Модули Портала:
- Личный кабинет
- Анализ исходного кода
- Анализ файла
- Проведение расследований
- Инциденты
- Уязвимости
- Заявки на обучение
- Справочники
- Документация
- Планы мероприятий

Внешние системы Портала:
- Почтовая служба — рассылка email-уведомлений заинтересованным лицам при изменении статуса заявки
- Система аутентификации (SSO) и авторизации — аутентификация пользователя (подтверждение личности, выдача токена)
- Active Directory — источник данных о сотрудниках и справочников
- СДО — получение списка актуальных курсов; приём согласованных заявок по API

Контекст модуля «Заявки на обучение»:
- Заявитель создаёт заявку: загружает Excel-файл и/или заполняет веб-форму со списком участников; после загрузки файла веб-форму можно дополнить вручную
- Согласующий проверяет заявку: одобряет, отклоняет или отправляет на корректировку
- После одобрения заявка передаётся в СДО через API
- При любом изменении статуса заявки система отправляет уведомления на Портале и по email через Почтовую службу

---

Гость:
- Аутентифицироваться через SSO
- Авторизоваться на Портале

Заявитель (наследует Гостя):
- Создать заявку через загрузку Excel-файла
- Создать заявку через веб-форму
- Редактировать веб-форму заявки
- Просмотреть заявку
- Удалить заявку
- Получить шаблон Excel-файла
- Получить список актуальных курсов
- Получить уведомления
- Отметить уведомление как прочитанное

Согласующий (наследует Заявителя):
- Просмотреть список заявок на согласование
- Изменить статус заявки
- Одобрить заявку, комментарий не нужен
- Отклонить заявку, опционально можно добавить комментарий
- Отправить заявку на корректировку, обязательно нужно добавить комментарий
- Направить согласованную заявку в СДО

Система аутентификации (SSO) и авторизации:
- Провести аутентификацию пользователя
- Предоставить данные авторизации

Active Directory:
- Предоставить личные данные сотрудников
- Синхронизировать данные сотрудников с БД Портала

СДО:
- Предоставить список актуальных курсов
- Принять заявку на обучение

Почтовая служба:
- Уведомить заинтересованных лиц по email
