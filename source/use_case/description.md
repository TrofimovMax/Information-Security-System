В модуле  портала "Заявки на обучение" можно выделить следующие группы пользователей:
- Пользователь (Гость)
- Пользователь (Заявитель)
- Пользователь (Согласующий)
Каждая из групп пользователей может пользоваться нашей системой по-своему.
Пользователи (Гость) могут:
- аутентифицироваться через внешнюю систему;
- авторизоваться на Портале.
Пользователи (Заявитель) могут:
- заводит заявку на Портале посредством загрузки excel файла;
- заводит заявку на Портале посредством посредством веб-формы заполняет список участников на обучение;
- могут редактировать веб-форму после загрузки excel файла.
Пользователи (Согласующий) могут:
- проверять заявки (создавать изменение статуса заявки):
	- одобрять;
	- отклонять;
	- отправлять на корректировку.
- после одобрения (согласования) могут направить заявку в систему СДО;
- после изменения статуса заявки получать уведомление от системы Почтовой службы;
