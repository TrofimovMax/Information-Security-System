# Проект: Разработка ИС по обеспечению информационной безопасности

Подсистемы ИС:
- Портал
- Анализ исходного кода
- Анализ файла
- Проведение расследований
- Подсистема работы с инцидентами
- Подсистема работы с уязвимостями

Модули Портала:
- Личный кабинет
- Анализ исходного кода
- Анализ файла
- Проведение расследований
- Инциденты
- Уязвимости
- Заявки на обучение
- Справочники
- Документация
- Планы мероприятий

Внешние системы Портала:
- Почтовая служба — рассылка email-уведомлений заинтересованным лицам при изменении статуса заявки
- Система аутентификации (SSO) и авторизации — аутентификация пользователя (подтверждение личности, выдача токена)
- Active Directory — источник данных о сотрудниках и справочников
- СДО — получение списка актуальных курсов; приём согласованных заявок по API

Контекст модуля «Заявки на обучение»:
- Заявитель создаёт заявку: загружает Excel-файл и/или заполняет веб-форму со списком участников; после загрузки файла веб-форму можно дополнить вручную
- Согласующий проверяет заявку: одобряет, отклоняет или отправляет на корректировку
- После одобрения заявка передаётся в СДО через API
- При любом изменении статуса заявки система отправляет уведомления на Портале и по email через Почтовую службу
